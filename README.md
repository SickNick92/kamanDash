Standalone Web Application Sample
=================================
Web application to display data from IBM's Watson IoT platform.

Dependency
==========
 - Nodejs
 - express
 - angularjs
 - ibmiotf
 - chart.js
 - angular-chart.js
 - bootstrap
 - jquery

Steps
======
1.Install the dependencies
  <pre>npm install</pre>
2.Start express application  
  <pre>npm start</pre>
3. open browser http://localhost:3000/
